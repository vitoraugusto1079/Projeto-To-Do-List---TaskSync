# Projeto To Do List - TaskSync

TaskSync é uma aplicação de gerenciamento de tarefas com funcionalidades de CRUD, desenvolvida para empresas controlarem projetos através de um quadro Kanban.

## Tecnologias Utilizadas
* HTML5, CSS3, JavaScript
* PHP (PDO)
* MySQL (XAMPP / phpMyAdmin)

## Como rodar o projeto localmente
1. Clone este repositório ou cole a pasta `TaskSync` dentro do diretório `htdocs` do seu XAMPP.
2. Inicie os módulos **Apache** e **MySQL** no Painel de Controle do XAMPP.
3. Aperte o botão "Admin" ao lado do Start do MySQL e importe o arquivo `tarefas.sql` (localizado na sub pasta "Estrutura_Banco") para gerar o banco de dados e as tabelas.
4. Acesse o sistema pelo navegador através da URL: `http://localhost/TaskSync/`

## Dependências
O sistema não requer a instalação de bibliotecas externas.
